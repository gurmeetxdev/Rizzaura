# RIZZ - Premium SaaS Landing Page

Built with **Next.js 14**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Project Structure

- `app/`: Next.js App Router (pages and layouts)
- `components/`: React components (Hero, DecodeSection, etc.)
- `lib/`: Utility functions and Rizz Analysis Engine
- `public/`: Static assets

## Features

- **Cosmic Background**: Interactive particle system.
- **Rizz Scanner**: Interactive charisma analysis tool.
- **Responsive Design**: Optimized for all screen sizes.
- **Glassmorphism**: Modern, premium aesthetic.
